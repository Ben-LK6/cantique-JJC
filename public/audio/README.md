# Dossier Audio des Cantiques

Placez vos fichiers audio ici avec le format : `cantique_XXX.mp3`

## Exemples :
- `cantique_001.mp3` pour le cantique n°001
- `cantique_002.mp3` pour le cantique n°002
- etc.

## Formats supportés :
- MP3 (recommandé)
- WAV
- OGG

## Instructions :
1. Enregistrez vos cantiques au format MP3
2. Nommez-les selon le numéro du cantique
3. Placez-les dans ce dossier
4. L'application les détectera automatiquement
